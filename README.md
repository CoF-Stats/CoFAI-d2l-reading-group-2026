# CoFAI D2L Reading Group - 2026

## Group Code of Conduct

### Inclusivity

- Respect different backgrounds and levels of expertise. AI and ML are jargon-filled fields, but don't assume the group knows the jargon. Define terms for the group before moving on.

- No one person or group of people should dominate conversations. We want to hear all voices.

- Save space for all questions. No question is too basic for this discussion.

### Discourse

- Ask clarifying questions before countering ideas. Listen to others assuming your understanding could in fact be incorrect.

- Keep discussion focused. Tangents related to ML and research projects are encouraged, but they should relate to the group's interests in this topic.

### Logistics

- Notes and notebooks will be stored on the group GitHub repo.

- Logistics, resources, and asynchronous discussions should take place on the group [Teams channel](https://teams.microsoft.com/l/channel/19%3Axmep0CksI5m0NiLy1AdVMDfXlk4qfPP57Co-Q08_q_I1%40thread.tacv2/D2L%20Reading%20Group?groupId=aa280d3f-344f-47a1-9a84-02c226934a35&tenantId=ce6d05e1-3c5e-4d62-87a8-4c4a2713c113&ngc=true&allowXTenantAccess=true)

- Coding and experimenting can be done on your local machine and separate git branch (or separate notebook), or on Google Colab.

    - To open a notebook from this repo in Colab, go to `colab.research.google.com → File → Open notebook → GitHub` and paste the repo URL. Alternatively, replace `github.com` in any notebook URL with `colab.research.google.com/github` for a direct link. **Note**: changes made in Colab won't save back to GitHub automatically — use `File → Save a copy in GitHub` to contribute changes.
